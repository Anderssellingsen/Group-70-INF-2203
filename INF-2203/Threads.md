
A thread is unit of execution within a process, however small that unit may be. A thread can for example be, a print function. There are two reasons for creating threads. First, a multithreaded CPU can be fully utilised to achieve parallelism. Finally, a thread is small and lightweight. Thus, threads can be destroyed much easier, and they can run faster than an entire process. Another property of threads is that they share the same memory address and resources.
